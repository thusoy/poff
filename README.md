poff [![Build Status](https://travis-ci.org/thusoy/poff.png)](https://travis-ci.org/thusoy/poff)
====

A plain old flask frontend for PowerDNS.

[Test coverage](https://thusoy.github.io/poff)

Development
-----------

Set up new virtualenv:

    $ virtualenv venv

Install everything:

    $ pip install -e .[test]

Run the tests:

    $ nosetests

Happy hacking!
